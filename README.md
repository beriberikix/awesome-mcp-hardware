# Awesome MCP for Hardware [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)
Awesome list of MCP servers & clients for interacting with hardware and the physical world.

> MCP (Model Context Protocol) is an open-source standard for connecting AI applications to external systems.

## Servers

- [tinymcp](https://github.com/golioth/tinymcp) - Let LLMs control embedded devices via the Model Context Protocol.
- [embedded-debugger-mcp](https://github.com/Adancurusul/embedded-debugger-mcp) - A MCP server for embedded debugging with probe-rs.
- [ros-mcp-server](https://github.com/robotmcp/ros-mcp-server) - Connect AI models like Claude & GPT with robots using MCP and ROS.
- [MCP_BluetoothScanner](https://github.com/parkerdorris/MCP_BluetoothScanner) - simple POC to test out local MCP tool usage through remote LLMs.
- [UnitApi](https://github.com/UnitApi/mcp) - A comprehensive library for secure hardware control through the Model Context Protocol (MCP), enabling AI agents and automated systems to interact with physical devices.
- [mcp2tcp](https://github.com/mcp2everything/mcp2tcp) - Conenct TCP device to LLMs via MCP protocol.
- [mcp2serial](https://github.com/mcp2everything/mcp2serial) - A open-source library enabling AI models to control hardware devices via serial communication using the MCP protocol. Initial support for Raspberry Pi Pico.
- [mcp2mqtt](https://github.com/mcp2everything/mcp2mqtt) - mcp2mqtt is a serial communication server based on the MCP service interface protocol, designed for communication with serial devices.

## Clients

- [robot-mcp-client](https://github.com/robotmcp/robot-mcp-client) - Connect AI models like Claude & GPT with robots using MCP and ROS.
